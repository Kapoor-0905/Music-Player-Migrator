Project Description

A web app that allows you to migrate your playlists and liked songs from one platform to the other with ease. It could potentially be integrated with the open website. The backend will be on python using the api's of the various music apps.


Purpose of Project

The project will help the user to easily transfer their favorite playlist to another platform which provides free music streaming. It will save the users time to find each and every song from the playlist and add them to the new playlist of another platform.


Project Features

  • The website will transfer playlists from Spotify to YouTube and vice versa.
  • User have to give the playlist link to transfer the playlist.
  • The playlist name and songs order will be preserved.
